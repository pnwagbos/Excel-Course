## 5 MINUTE SETUP

### STEP 1: Create GitHub Account (2 minutes)

1. Go to **https://github.com**
2. Click **"Sign up"** (top right)
3. Enter:
   - Email address
   - Password
   - Username (e.g., `your-name`)
4. Verify your email
5. Done! You have a GitHub account

### STEP 2: Create a New Repository (1 minute)

1. Click **"+"** button (top right)
2. Select **"New repository"**
3. Repository name: **`excel-function-course`**
4. Description: Excel Function Course
5. **Initialize with README** - Check this box
6. Click **"Create repository"**
7. Done! You have a repository

### STEP 3: Upload Your Files (1 minute)

1. Click **"Add file"** button
2. Select **"Upload files"**
3. **Drag and drop** all files from `C:\tmp` into the upload area
4. OR click **"choose your files"** and select them
5. Scroll down
6. Click **"Commit changes"**
7. Wait for upload to complete
8. Done! Files are uploaded

### STEP 4: Enable GitHub Pages (1 minute)

1. Go to **Settings** tab (top of repository)
2. Click **"Pages"** in the left menu
3. Under "Source", select:
   - Branch: **main**
   - Folder: **/ (root)**
4. Click **"Save"**
5. Wait 30 seconds
6. You'll see: "Your site is published at: https://your-username.github.io/excel-function-course"

### STEP 5: Share Your Course! (Done!)

1. Copy the URL from Step 4
2. Send to students!
3. They click the link
4. Your course opens!
5. No installation needed!

## TROUBLESHOOTING:

- Clear browser cache
- Check URL spelling

### Files not uploaded?
- Check you uploaded at least `index.html`

- Go to Settings tab
